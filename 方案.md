# Novel Voice Cast — 开发方案

## 背景

我喜欢看轻小说，但轻小说的对话通常不标注说话人，只用「」包裹。市面上的 TTS 工具要么只有单一朗读者，要么配置复杂、需要自己折腾音色克隆。

这个项目的目标很简单：**给定一本已标注好说话人的小说 + config 配置 → 一键合成带角色演绎的有声书**。

## 项目结构

```

│   │   │   ├── emotion_labeler.py # 情感标注
│   │   │   ├── ollama_client.py   # Ollama 客户端
│   │   │   └── gender_identifier.py # 性别识别
│   │   └── tts/
│   │       ├── base.py            # TTS 基类
│   │       ├── preset.py          # edge-tts
│   │       └── voice_clone.py     # VoxCPM
│   ├── pipeline.py           # 核心流程
│   └── denoise.py            # 音频去噪
├── novels/                   # 小说文件
├── voices/                   # 角色参考音频
└── output/                   # 输出目录
```

## config.yaml 示例

```yaml
# 小说文件
novel:
  text_path: "novels/novel.txt"        # 小说文本
  labels_path: "novels/labels.txt"     # 角色标注

# Ollama 配置（用于性别识别、情感标注）
ollama:
  base_url: "http://172.31.102.237:11434"
  model: "qwen3:4b"

# 角色音频配置
characters:
  # 用 VoxCPM 克隆的角色
  - name: "罗伦斯"
    voice: "voxcpm"
    reference_audio: "voices/ronald.wav"
  - name: "赫萝"
    voice: "voxcpm"
    reference_audio: "voices/holo.wav"
  
  # 其他角色自动用预设（根据性别）

# 功能开关
features:
  gender_identify: true        # 自动识别性别
  emotion_label: true          # 情感标注
  denoise: true                # 音频去噪

# 输出配置
output:
  dir: "output"
  filename: "full_volume.wav"
```

## 流程说明

```
┌─────────────────────────────────────────────────────────┐
│  run.py                                                 │
│  ├── 1. 解析小说 (parser.py)                             │
│  │      novel.txt + labels.txt → 1287条对话, 22个角色    │
│  │                                                       │
│  ├── 2. 识别性别 (gender_identifier.py) [可选]           │
│  │      Ollama 分析 → 保存到 gender_results.json         │
│  │                                                       │
│  ├── 3. 情感标注 (emotion_labeler.py) [可选]             │
│  │      Ollama 分析 → 保存到 emotion_results.json        │
│  │                                                       │
│  ├── 4. TTS 合成 (voice_clone.py / preset.py)           │
│  │      VoxCPM: 罗伦斯、赫萝 (约30s/条)                  │
│  │      edge-tts: 其他角色 (约0.5s/条)                   │
│  │      → 1287个音频片段                                 │
│  │                                                       │
│  ├── 5. 音频拼接 (splicer.py)                            │
│  │      按顺序拼接 + 间隔 + 淡入淡出                      │
│  │      → full_volume.wav                                │
│  │                                                       │
│  └── 6. 去噪 (denoise.py) [可选]                         │
│         DeepFilterNet3                                   │
│         → full_volume_denoised.wav                       │
└─────────────────────────────────────────────────────────┘
```

## 技术选型

| 模块 | 选型 | 理由 |
|------|------|------|
| 配置 | **YAML** | 人类可读，支持注释 |
| 音频处理 | **pydub** + **soundfile** | 格式转换、拼接 |
| 音色克隆 | **VoxCPM** | 中文效果好 |
| 预设 TTS | **edge-tts** | 免费，中文自然 |
| 性别/情感 | **Ollama** (qwen3:4b) | 本地运行 |
| 去噪 | **DeepFilterNet3** | 效果好，轻量 |

## 运行方式

```bash
# 1. 编辑配置
vim config.yaml

# 2. 运行
python run.py

# 或指定配置文件
python run.py --config my_config.yaml
```
